detect_outlier <- function(x) {
  Q1 <- quantile(x, 0.25, na.rm = TRUE)
  Q3 <- quantile(x, 0.75, na.rm = TRUE)
  IQR <- Q3 - Q1
  lower_bound <- Q1 -1.5 * IQR
  uper_bound <- Q3 +1.5 * IQR
  return(x < lower_bound | x > uper_bound)
}

cols_to_check = c("length_cm", "price", "weight_g")

for (col in cols_to_check) {
  outliers <- detect_outlier(zoomba_data[[col]])
  num_outliers <- sum(outliers, na.rm = TRUE)
  
  cat("\nColumn:", col, "\n")
  cat("Number of outliers:", num_outliers, "\n")
  
  if (num_outliers > 0) {
    mean_val <- mean(zoomba_data[[col]][!outliers], na.rm = TRUE)
    zoomba_data[[col]][outliers] <- mean_val
    cat("Outliers replaced with mean.\n")
  } else {
    cat("No outliers found.\n")
  }
}

Develop a model to predict the price of a Zoomba fruit. You can use the models we learned in class but not anything else. You are allowed to employ clustering, dependency modelling, decision tree models or SVM. You are not allowed to use any models beyond that, such as random forest or ANN.

Explanation

A SVM model is developed and fine-tuned across different parameters such as cost, gamma and epsilon values. Best model has been choosen according to lowest RMSE value. Dataset is divided into train and validation parts for the model by 80-20 ratio.

Answer

set.seed(456)

n <- nrow(zoomba_data)
train_index <- sample(1:n, size = 0.8 * n)

train_data <- zoomba_data[train_index, ]
val_data <- zoomba_data[-train_index, ]

cost_values <- c(0.1, 1, 10)
gamma_values <- c(0.1, 0.25, 0.5)
epsilon = c(0.01, 0.1, 0.2)

best_model <- NULL
lowest_rmse <- Inf

for (c in cost_values) {
  for (g in gamma_values) {
    for (e in epsilon) {
      model <- svm(price ~ ., data = train_data, type = "eps-regression", cost = c, gamma = g, epsilon= e)
      preds <- predict(model, val_data)
      rmse <- sqrt(mean((preds - val_data$price)^2))
      
      if (rmse < lowest_rmse) {
        lowest_rmse <- rmse
        best_model <- model
      }
    }
    
  }
}

my_model  <- best_model
